# Bloom

Bloom is a full-stack banking demo application with a Next.js frontend and an Express backend.

## Tech Stack

### Frontend

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Base UI (`@base-ui/react`)
- Recharts
- Sonner
- Lucide React
- NextAuth v5 beta

### Backend

- Node.js
- Express 5
- TypeScript
- Prisma ORM

### Database

- PostgreSQL

### Authentication

- Google OAuth via NextAuth

## Project Structure

- `frontend/` - Next.js application
- `backend/` - Express API and Prisma schema
